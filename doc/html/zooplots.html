Zooplots¶

A zoo plot shows a scatter type plot where on the vertical axis the mean of the non target scores and on the horizontal axis the mean target scores are drawn. This leads to a plot of dots where each dot represents one speaker ( assuming the data stems from a speaker id experiment ). An extention of the zoo plots was shown at the IAFPA 2014 conference in Zurich, Switserland by Anil Alexander et al. He proposed that adding a measure of the standard deviations of the scores used to make the plot will add details of the score distributions of the persons to the plot. If the option alexanderStyle in [zoo] is set to True, ellipses are drawn at the positions where the points of a traditional zoo plot would be. The width and height of the ellipses shown are essentially the standard deviations of the average target and average non target scores for a given label. Because these may be much bigger or much smaller than the horizontal and vertical scales of the traditional zoo plot, the mean standard deviations are scaled by subtracting the overall mean standard deviation and dividing by the standard deviation of all standard deviations. This is in essence a normalization procedure. The result will be ellipses with a unit width and height and ellipses smaller and bigger than that. To be able to actually plot the normalised ellipses, the width is multiplied by the range of scores on the horizontal axis and the height is multiplied by the range of the scores on the vertical axis. Finally to scale the ellipses their width and height is divided by a scale factor. This scale factor is related to the number of pixels in the display used to plot the zoo plot. A value of 150 works nicely for a 1600 ... 1280x1024 display.

Note that the shape of the ellipses is mainly due to the difference in range of the vertical and horizontal axis. This means that comparing shapes between zoo plots with varying ranges of mean target and mean non target scores can be very tricky.

Highlighting labels¶

If you click on a data point in the plot, a text label will be shown near the point. This makes it possible to find the name of a data point in e.g. the quartile ranges.

If you are curious where the scores of a specific label are in the zoo plot, you need not click on all of them to find it. You can specify the labels on the command line. If they are in the plot, they will be highlighted. Example:

python ./bioplot.py -e “condition A” -f testdata_A.txt -Z 1130 1062 226

This will highlight label 1130, 1062 and 226 in the zoo plot compiled from the data in ‘testdata_A.txt’ and dim the colors of the other points in the plot making it easy to create a picture for a publication or report. Text labels will be displayed near the points selected.

If you want to see the combined results of experiments with the same dataset but with some change in parameters, try the dataset which combines the ‘A’, ‘B’ and ‘C’ experiment:

python ./bioplot.py -e “exp A vs B vs C” -f testdata_ABC.txt -Z

The lines between the ellipses connect labels which are equal. This makes it easy to see what the effect of the parameter change is.

If you click on a data point it’s label will be shown.
